# git-task

## Description

The **git-task** project is a practical exercise focused on learning and applying fundamental Git version control concepts. The task demonstrates the process of creating and managing a Git repository while tracking changes made to files throughout development.

The project includes a simple program (`hello_world.py`) that initially prints **"Hello World!"** and is later modified to print **"Git is awesome!"** to demonstrate how Git tracks file changes and versions.

## Objectives

This task covers the following Git concepts:

- Initialising a Git repository using `git init`
- Checking repository status using `git status`
- Tracking files with `git add`
- Staging modified files
- Creating commits using `git commit`
- Viewing commit history with `git log`
- Understanding the Git workflow:
  - Untracked files
  - Staged changes
  - Committed changes

## Files Included

- `hello_world.py` – Python program used to demonstrate Git tracking and versioning
- `gitlog.png` – Screenshot of Git commit history output from `git log`
